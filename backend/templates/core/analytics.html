Ejemplo combinado (ORM)

Misma lógica que “todos los lunes de enero de 2026”: suma de monto con año + mes + día fijos en código. Si el dataset no tiene ese mes, el total es 0.

Suma monto · Lunes + Enero + 2026: ${{ ej_orm_lunes_enero_2026 }}

Referencia SQL (DBeaver / psql)

Equivalente a los joins que el ORM hace entre core_factventas y core_dimtiempo: 

SELECT SUM(f.monto) AS total_monto
FROM core_factventas f
JOIN core_dimtiempo t ON f.fecha_id = t.id_tiempo
WHERE t.nombre_dia = 'Lunes'
  AND t.nombre_mes = 'Enero'
  AND t.anio = 2026;
